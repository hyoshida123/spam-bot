# 4行のスパムボット
## 環境設定
Terminalを開いて、
`virtualenv spam-bot`
で仮想環境を構築します。virtualenvがインストールされてない場合は、
`pip install virtualenv`
でインストールできます。
`virtualenv spam-bot/bin/activate`
で仮想環境を起動したら、
`pip install pyautogui`
で必要なライブラリをインストールします。

## 使い方
`python3 spam-bot.py`
で起動したら、LINEやYoutubeライブのテキスト入力ボックスをクリックするだけです。
